# Drawing App

Drawing app developed in kotlin.


![Logo](https://i.imgur.com/b7GLYCg.png)


## Tools and Languages

- Kotlin
- Lottie Animation
- Color picker
- Android Studio
- Git, Github


## Features

- Draw on different brush sizes
- Change paint color
- Use images for background
- Undo and redo drawing
- Save drawing as png file
